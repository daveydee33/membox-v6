# MemBox (membox5)
A fullstack javascript application created for language learners

# Getting Started
* copy `.env.example` to `.env`, and set the required values
* start app
  * `npm run dev` for Development
  * `npm start` for Production

# API Routes
* /items
  * GET - get ALL items
  * POST - create one item
